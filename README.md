
# База знаний

---

### **1. `/IT/Developer/FullStack/RU/`**
**Описание:**
Основная папка с материалами на русском языке, предназначенная для FullStack-разработчиков. Содержит общие гайды, дорожные карты, рекомендации и ссылки на ресурсы.

**Содержимое:**
- `README.md` — общий обзор репозитория.
- `RoadMap.md` — дорожная карта изучения технологий.
- `GuidLines.md` — рекомендации и лучшие практики.
- `Links.md` — полезные ссылки на статьи, курсы и инструменты.

---

### **2. `/Docs/`**
**Описание:**
Техническая документация по различным технологиям, включая C#, Docker, Kubernetes, ELK, Git, базы данных, очереди сообщений, OOP, Python и тестирование.

**Содержимое:**
- Папки по каждой технологии (например, `/Docs/C#/`, `/Docs/Docker/`).
- Теоретические материалы, примеры кода, конфигурационные файлы.

---

### **3. `/Code/`**
**Описание:**
Предназначена для практических примеров кода. 

---

### **4. `/Prompts/`**
**Описание:**
Запросы для LLM (Large Language Models) — например, для генерации кода или документов.

---

### **5. `/Questions/`**
**Описание:**
Вопросы с собеседований (технические и теоретические).

### **6. `/TestTasks/`**

### **7. `/Vacancies/`**

### **8. `/RoadMaps/`**
