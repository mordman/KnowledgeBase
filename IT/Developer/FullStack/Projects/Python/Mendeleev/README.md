# 📚 Интерактивная Таблица Менделеева
## Описание приложения и руководство пользователя

---

## 🎯 О приложении

**Интерактивная Таблица Менделеева** — это графическое приложение на Python, которое предоставляет удобный и наглядный способ изучения химических элементов. Приложение построено исключительно на стандартных библиотеках Python (tkinter, json), что обеспечивает лёгкость установки и запуска без необходимости подключения сторонних зависимостей.

Каждый элемент таблицы представлен в виде интерактивной карточки с цветовой кодировкой по категориям, анимацией при наведении и подробной информацией при клике.

---

## ✨ Основные возможности

| № | Возможность | Описание |
|---|-------------|----------|
| 1 | 📊 **Полная таблица элементов** | Все 118 химических элементов с точным расположением по периодам и группам |
| 2 | 🎨 **Цветовая категоризация** | Каждая категория элементов имеет уникальный цвет (металлы, неметаллы, газы и т.д.) |
| 3 | 🖱️ **Интерактивные карточки** | При наведении мыши элемент увеличивается, при клике — открывается подробная информация |
| 4 | 📝 **HTML-описания** | Поддержка форматированного текста с тегами `<b>`, `<i>`, `<u>`, списками |
| 5 | 🧪 **Молекулы и формулы** | Отдельное окно с примерами химических соединений и их формулами |
| 6 | 📋 **Легенда категорий** | Наглядное отображение всех категорий с цветовыми обозначениями |
| 7 | ⚙️ **Гибкие настройки** | Изменение размеров ячеек, окна приложения и цветов категорий через интерфейс |
| 8 | 💾 **JSON-конфигурация** | Все данные и настройки хранятся в отдельных файлах для лёгкого редактирования |
| 9 | 🔄 **Адаптивный интерфейс** | Все окна можно изменять в размерах, содержимое автоматически подстраивается |
| 10 | 📜 **Прокрутка таблицы** | Удобная навигация по всей таблице с горизонтальной и вертикальной прокруткой |

## 🚀 Установка и запуск

### Шаг 1: Установка Python
Убедитесь, что у вас установлен Python 3.6 или выше:
```bash
python --version
```

### Шаг 2: Создание файлов
1. Создайте папку для проекта (например, `Mendeleev`)
2. Сохраните все файлы (`main.py`, `elements.json`, `molecules.json`, `settings.json`) в эту папку

### Шаг 3: Запуск приложения
```bash
cd Mendeleev
python main.py
```

### Шаг 4: Первый запуск
При первом запуске приложение автоматически создаст отсутствующие JSON-файлы с данными по умолчанию.

## ⚙️ Настройки приложения

Файл `settings.json` содержит следующие параметры:

| Параметр | Тип | Описание | По умолчанию |
|----------|-----|----------|--------------|
| `cell_width` | int | Ширина ячейки элемента (пиксели) | 65 |
| `cell_height` | int | Высота ячейки элемента (пиксели) | 75 |
| `window_width` | int | Ширина главного окна (пиксели) | 1200 |
| `window_height` | int | Высота главного окна (пиксели) | 850 |
| `category_colors` | object | Цвета для каждой категории элементов | См. ниже |

## 📝 Формат файлов данных

### elements.json

Каждый элемент имеет следующую структуру:

```json
{
    "number": 1,
    "symbol": "H",
    "name": "Водород",
    "mass": 1.008,
    "category": "Неметаллы",
    "row": 1,
    "col": 1,
    "description": "Краткое описание элемента",
    "html": "<b>Форматированное</b> описание с тегами"
}
```

| Поле | Тип | Обязательное | Описание |
|------|-----|--------------|----------|
| `number` | int | ✅ | Атомный номер элемента |
| `symbol` | string | ✅ | Химический символ (1-2 буквы) |
| `name` | string | ✅ | Полное название элемента |
| `mass` | float | ✅ | Атомная масса |
| `category` | string | ✅ | Категория (должна совпадать с settings.json) |
| `row` | int | ✅ | Номер периода (1-7, 8-9 для лантаноидов/актиноидов) |
| `col` | int | ✅ | Номер группы (1-18) |
| `description` | string | ❌ | Простое текстовое описание |
| `html` | string | ❌ | HTML-форматированное описание (приоритет) |

> ⚠️ **Важно:** Если указано поле `html`, поле `description` игнорируется, и дополнительная информация (атомный номер, масса и т.д.) не отображается.

### molecules.json

```json
[
    {
        "name": "Вода",
        "formula": "H2O",
        "description": "Оксид водорода, основа жизни"
    }
]
```

## 🎨 HTML-форматирование описаний

Приложение поддерживает базовые HTML-теги для форматирования текста в описаниях элементов.

### Поддерживаемые теги:

| Тег | Описание | Пример |
|-----|----------|--------|
| `<b>текст</b>` | **Жирный текст** | `<b>Важно</b>` |
| `<i>текст</i>` | *Курсив* | `<i>примечание</i>` |
| `<u>текст</u>` | <u>Подчёркивание</u> | `<u>термин</u>` |
| `<br>` | Перенос строки | `строка 1<br>строка 2` |
| `- текст` | Маркированный список | `- Первый пункт` |
| `1. текст` | Нумерованный список | `1. Первый пункт` |

### Пример HTML-описания:

```json
"html": "<b>Углерод</b> — основа органической химии.\n\n<i>Аллотропные формы:</i>\n- Алмаз\n- Графит\n- Фуллерен\n- Графен"
```

### Результат отображения:

```
Углерод — основа органической химии.

Аллотропные формы:
• Алмаз
• Графит
• Фуллерен
• Графен
```

## 🔧 Расширение функционала

### Добавление нового элемента:

1. Откройте `elements.json`
2. Добавьте новую запись в массив:
```json
{
    "number": 119,
    "symbol": "Uue",
    "name": "Унуненний",
    "mass": 315,
    "category": "Щелочные металлы",
    "row": 8,
    "col": 1,
    "description": "Гипотетический элемент"
}
```
3. Перезапустите приложение

### Изменение цвета категории:

1. Откройте `settings.json`
2. Измените значение в `category_colors`:
```json
"Щелочные металлы": "#FF0000"
```
3. Перезапустите приложение или сохраните через интерфейс настроек

### Добавление новой молекулы:

1. Откройте `molecules.json`
2. Добавьте новую запись:
```json
{
    "name": "Аспирин",
    "formula": "C9H8O4",
    "description": "Лекарственное средство"
}
```
3. Перезапустите приложение

## 🐛 Решение проблем

| Проблема | Решение |
|----------|---------|
| Приложение не запускается | Проверьте версию Python (должна быть 3.6+) |
| Элементы не отображаются | Проверьте файл `elements.json` на корректность JSON |
| Цвета не применяются | Проверьте названия категорий в `elements.json` и `settings.json` |
| Окно слишком маленькое | Измените `window_width` и `window_height` в настройках |
| HTML не отображается | Проверьте правильность закрывающих тегов |
