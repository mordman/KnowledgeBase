## 🎯 О проекте

**Интерактивная Таблица Менделеева** — это образовательное приложение с открытым исходным кодом, разработанное на Python с использованием стандартной библиотеки Tkinter. Приложение предоставляет удобный интерфейс для изучения химических элементов с возможностью полной кастомизации внешнего вида через систему шаблонов.

### Ключевые особенности:
- ✅ **118 химических элементов** с полными данными
- ✅ **7 готовых шаблонов** оформления (мини, стандарт, крупный, тёмный, светлый, цветной, пастельный)
- ✅ **Markdown-поддержка** для химических формул (H₂O, CO₂, H⁺, OH⁻)
- ✅ **Полная кастомизация** шрифтов, цветов и размеров
- ✅ **Только стандартные библиотеки** Python (без внешних зависимостей)

## 📄 Форматы данных

### elements.json

```json
{
    "number": 1,
    "symbol": "H",
    "name": "Водород",
    "mass": 1.008,
    "category": "Неметаллы",
    "row": 1,
    "col": 1,
    "description": "Краткое описание",
    "markdown": "**Водород** — первый элемент.\n\nФормула: H_2_O",
    "html": "<b>Водород</b> — первый элемент."
}
```

| Поле | Тип | Обязательное | Описание |
|------|-----|--------------|----------|
| `number` | int | ✅ | Атомный номер |
| `symbol` | string | ✅ | Химический символ |
| `name` | string | ✅ | Название элемента |
| `mass` | float | ✅ | Атомная масса |
| `category` | string | ✅ | Категория (должна совпадать с settings.json) |
| `row` | int | ✅ | Номер периода (1-9) |
| `col` | int | ✅ | Номер группы (1-18) |
| `description` | string | ❌ | Простое описание |
| `markdown` | string | ❌ | Markdown-описание (приоритет) |
| `html` | string | ❌ | HTML-описание |

> ⚠️ **Приоритет полей:** `markdown` → `html` → `description`

### molecules.json

```json
[
    {
        "name": "Вода",
        "formula": "H_2_O",
        "description": "Оксид водорода"
    }
]
```

### settings.json

```json
{
    "cell_width": 65,
    "cell_height": 75,
    "window_width": 1200,
    "window_height": 850,
    "current_template": "standard",
    "fonts": {...},
    "colors": {...},
    "category_colors": {...},
    "templates": {...}
}
```

## Markdown-формулы

### Поддерживаемый синтаксис

| Синтаксис | Результат | Описание |
|-----------|-----------|----------|
| `H_2_O` | H₂O | Подстрочный индекс |
| `H^+^` | H⁺ | Надстрочный индекс |
| `SO_4^2-^` | SO₄²⁻ | Комбинированный |
| `**жирный**` | **жирный** | Жирный текст |
| `*курсив*` | *курсив* | Курсив |
| `- пункт` | • пункт | Маркированный список |
| `1. пункт` | 1. пункт | Нумерованный список |
| `# Заголовок` | Заголовок | Заголовок |

### Примеры использования

```json
{
    "markdown": "**Углерод** — основа органики.\n\nФормулы:\n- CO_2^ — углекислый газ\n- C_6_H_12_O_6^ — глюкоза\n- CH_4^ — метан"
}
```

**Результат:**

```
Углерод — основа органики.

Формулы:
• CO₂ — углекислый газ
• C₆H₁₂O₆ — глюкоза
• CH₄ — метан
```

## 🎨 Создание своих шаблонов

### Через интерфейс

1. Настройте все параметры в **⚙️ Настройки**
2. Перейдите на вкладку **📋 Шаблоны**
3. Нажмите **💾 Сохранить текущие как шаблон**
4. Введите название и описание
5. Шаблон появится в списке

### Через settings.json

```json
"templates": {
    "my_template": {
        "name": "Мой шаблон",
        "description": "Описание шаблона",
        "fonts": {...},
        "colors": {...},
        "category_colors": {...},
        "cell_width": 70,
        "cell_height": 80
    }
}
```

### Удаление шаблона

- **Системные шаблоны** (mini, standard, large, dark, light, colorful, pastel) — нельзя удалить
- **Пользовательские шаблоны** — можно удалить через интерфейс или вручную в `settings.json`

## 🔧 Расширение функционала

### Добавление нового элемента

1. Откройте `data/elements.json`
2. Добавьте запись:

```json
{
    "number": 119,
    "symbol": "Uue",
    "name": "Унуненний",
    "mass": 315,
    "category": "Щелочные металлы",
    "row": 8,
    "col": 1,
    "description": "Гипотетический элемент"
}
```

3. Перезапустите приложение

### Добавление новой категории

1. Откройте `settings.json`
2. Добавьте цвет в `category_colors`:

```json
"category_colors": {
    "Новая категория": "#FF0000"
}
```

3. Укажите эту категорию в `elements.json`

### Добавление нового шрифта

Откройте `settings.json` и добавьте в `fonts`:

```json
"fonts": {
    "new_font": {"family": "Verdana", "size": 12, "weight": "bold"}
}
```

## 🐛 Решение проблем

| Проблема | Возможная причина | Решение |
|----------|-------------------|---------|
| **Приложение не запускается** | Python < 3.6 | Обновите Python до 3.6+ |
| **Элементы не отображаются** | Ошибка в elements.json | Проверьте JSON на валидность |
| **Цвета не применяются** | Несоответствие категорий | Проверьте названия в elements.json и settings.json |
| **Формулы не рендерятся** | Ошибка в синтаксисе | Используйте H_2_O, а не H2O |
| **Шаблоны не работают** | Повреждён settings.json | Удалите файл, он создастся заново |
| **Окно слишком маленькое** | Настройки размеров | Измените window_width/window_height в настройках |
| **Шрифты не меняются** | Шрифт не установлен в ОС | Используйте стандартные шрифты (Arial, Times) |
| **Прокрутка не работает** | Элементы за пределами | Увеличьте размеры окна или уменьшите ячейки |
