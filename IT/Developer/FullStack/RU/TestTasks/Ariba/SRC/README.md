## Solution

В данном разделе опишите архитектуру решения, используемые технологии, основные компоненты и инструкции по запуску проекта.

### Используемые технологии
- .NET 6
- ASP.NET Core
- Entity Framework Core
- React

### Запуск проекта
1. Откройте решение в Visual Studio 2022.
2. Выполните сборку проекта.
3. Запустите проект с помощью IIS Express или Kestrel.

### Структура проекта
- `SRC/Backend` — серверная часть (API)
- `SRC/Frontend` — клиентская часть (React)
