### **Проекты Apache Software Foundation (ASF)**

Apache Software Foundation поддерживает сотни проектов с открытым исходным кодом, охватывающих веб-серверы, Big Data, базы данных и инструменты разработки. Наиболее популярные проекты — HTTP Server, Kafka, Spark, Airflow и Cassandra.

---

### 🖥️ **Веб-серверы и инфраструктура**

- **Apache HTTP Server (httpd)**  
  Самый популярный в мире веб-сервер, обеспечивающий кроссплатформенность, надежность и гибкую настройку через модули.

- **Apache Tomcat**  
  Контейнер сервлетов Java (Java Servlet Container), используемый для запуска Java-приложений (JSP, Java EE), работает как самостоятельный веб-сервер.

---

### 📊 **Big Data и потоковая обработка**

- **[Apache Kafka](/Docs/Appache/Kafka/README.md)**  
  Распределенная платформа потоковой передачи событий (event streaming), используемая для передачи данных в реальном времени, обработки потоков и интеграции систем.

- **[Apache Spark](/Docs/Appache/Spark/README.md)**  
  Движок для быстрой обработки данных, поддерживающий SQL-запросы, потоковые данные (Streaming), машинное обучение (MLlib) и графовые вычисления.

- **Apache Hadoop**  
  Базовая платформа для хранения (HDFS) и обработки (MapReduce) огромных наборов данных на кластерах стандартного оборудования.

- **Apache Cassandra**  
  Высокопроизводительная распределенная система управления базами данных (NoSQL), созданная для обработки больших объемов данных по множеству серверов без единой точки отказа.

---

### ⚙️ **Оркестрация и ETL**

- **[Apache Airflow](/Docs/Appache/AirFlow/README.md)**  
  Платформа для создания, планирования и мониторинга рабочих процессов (workflow) и ETL-пайплайнов с помощью Python-кода (DAG).

- **[Apache NiFi](/Docs/Appache/NiFi/README.md)**  
  Инструмент для автоматизации потоков данных (DataFlow) между системами, поддерживающий визуальное управление маршрутизацией, трансформацией и медиацией данных.

---

### 🛠️ **Инструменты разработчика и аналитика**

- **Apache Maven**  
  Инструмент для автоматизации сборки Java-проектов, управления зависимостями и документацией.

- **Apache Zeppelin**  
  Веб-блокнот (notebook) для интерактивной аналитики данных, позволяющий работать с SQL, Spark и другими технологиями в единой среде.

- **Apache Superset**  
  Современная платформа для визуализации данных и создания аналитических дашбордов, поддерживающая множество баз данных.
