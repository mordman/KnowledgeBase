## Описание

- Один HTML-файл, без зависимостей
- ES6+ классы, `requestAnimationFrame`
- Поле 10×20 с адаптивным размером клеток
- 100 уровней с прогрессивной скоростью
- Переход по количеству линий (настраивается)
- Мягкое/жёсткое падение, вращение с wall-kick
- Ghost piece (призрак)
- Подсчёт очков с множителем уровня 
- localStorage + валидация данных
- Экспорт/импорт JSON с try/catch
- Редактор уровней (3 вкладки)
- Настройка вероятностей фигур с нормализацией
- Индикаторы сложности (формула из 4 компонентов)
- Неоновая палитра с `box-shadow`/`text-shadow`
- Модальные окна без перезагрузки
- Комментарии на русском

**Управление:** `← →` — движение, `↑` — вращение, `↓` — мягкое падение, `Пробел` — жёсткое падение.
