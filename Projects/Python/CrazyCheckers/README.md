# Сетевые Шашки на Python v2.0

## Структура проекта

```
checkers/
├── main.py              # Запуск игры
├── config.py            # Управление конфигурацией
├── settings.py          # Константы по умолчанию
├── network.py           # Сетевая логика
├── board.py             # Логика игры
├── ui.py                # Графический интерфейс
├── dialogs.py           # Диалоговые окна
├── config.json          # Настройки (создаётся автоматически)
└── README.md            # Этот файл
```

## Запуск
```bash
python main.py
```

## Настройки
**Игра → Настройки** позволяет изменить:
- Размер доски (6-10 клеток)
- Размер клетки (40-120 px)
- Размер окна
- Размер шашек (масштаб 0.5-0.9)
- Все цвета (клетки, шашки, подсветка)

Настройки сохраняются в `config.json` и применяются после перезапуска.

## Сетевая игра
1. **Хост**: Игра → Сетевая игра → Создать игру
2. **Клиент**: Игра → Сетевая игра → Присоединиться

## Правила
- Русские шашки
- Рубка обязательна
- Рубка назад разрешена
- Серийные удары поддерживаются
