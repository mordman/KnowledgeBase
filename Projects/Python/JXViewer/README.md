# JSON/XML Viewer

Просмотрщик JSON и XML файлов с подсветкой синтаксиса и поиском.

## Запуск

```bash
python main.py
```

## Горячие клавиши

- `Ctrl+O` — Открыть файлы
- `Ctrl+F` — Фокус на поиск
- `Ctrl+W` — Закрыть вкладку
- `F3` — Следующий результат поиска

## Структура

- `config.py` — Настройки
- `parser.py` — Парсеры
- `widgets.py` — Виджеты
- `viewer.py` — Логика UI
- `main.py` — Точка входа
```

---

### Преимущества такой структуры

| Файл | Ответственность |
|------|-----------------|
| `config.py` | Все настройки в одном месте |
| `parser.py` | Логика парсинга изолирована |
| `widgets.py` | Переиспользуемые компоненты |
| `viewer.py` | Основная логика приложения |
| `main.py` | Чистая точка входа |
